# NextAV -- Next Artificial View

**AI-powered geospatial intelligence for critical infrastructure monitoring.**

NextAV transforms satellite imagery into actionable insights using super-resolution AI, enabling operators of pipelines, power networks, and industrial facilities to monitor thousands of kilometers of infrastructure from space.

## Products

- **SRView** -- AI super-resolution imagery (Sentinel-2 10m to 1m) for high-fidelity monitoring
- **MethaneView** -- Satellite-based methane emission detection and tracking for oil & gas
- **GreenView** -- Vegetation health and land cover classification for utilities and agriculture

## Technology

- **AI/ML:** PyTorch super-resolution models, Temporal Fusion Transformers
- **Geospatial:** Sentinel-2, Cloud Optimized GeoTIFFs, WMTS/WMS, PostGIS
- **Platform:** FastAPI, React, TerriaJS, PostgreSQL, AWS (EKS, RDS, S3, CloudFront)
- **Infrastructure:** Terraform, Docker, GitHub Actions
