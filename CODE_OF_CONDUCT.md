# Code of Conduct

NextAV is committed to maintaining a professional, secure, and collaborative environment for all team members, contractors, and partners interacting with our codebase and infrastructure.

## Professional Standards

All contributors are expected to:

- **Quality and rigor:** Write clean, tested, documented code. Engage in thorough and constructive code reviews.
- **Respectful communication:** Maintain a professional tone in all pull requests, issues, and communications.
- **Security first:** Protect intellectual property, client data, and production systems. Follow established security protocols.
- **Ownership:** Take responsibility for your work. If you see a problem, flag it or fix it.

## Unacceptable Behavior

The following behaviors will not be tolerated:

- Harassment, discrimination, or unprofessional conduct of any kind
- Sharing, exposing, or mishandling proprietary code, trade secrets, or client data
- Disregarding established security protocols and engineering guidelines
- Committing credentials, API keys, or secrets to version control

## Enforcement

Violations of this code of conduct, particularly those regarding security and intellectual property, will result in immediate revocation of repository access and may lead to termination of contract.

To report a concern, contact leadership directly at [contact@nextav.ai](mailto:contact@nextav.ai).
